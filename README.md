# Data Processing Algorithms - Practice Repository

Interactive learning area to practice data processing algorithm related topics.

This repository includes applications and practice exercises related to the Istanbul Technical University Mathematics Engineering **MAT 333/E** course.

## Course Structure

This repository is organized by topics for easier study:

1. **[Basic Concepts](./01-basic-concepts/)** - Introduction to algorithms and analysis
2. **[Stable Matching](./02-stable-matching/)** - Gale-Shapley algorithm
3. **[Complexity Analysis](./03-complexity-analysis/)** - Asymptotic analysis and running times
4. **[Graphs](./04-graphs/)** - Graph algorithms and traversal
5. **[Greedy Algorithms](./05-greedy-algorithms/)** - Greedy problem-solving techniques
6. **[Divide and Conquer](./06-divide-and-conquer/)** - Divide and conquer strategies
7. **[Dynamic Programming](./07-dynamic-programming/)** - Dynamic programming techniques
8. **[Network Flow](./08-network-flow/)** - Flow algorithms and applications
9. **[Complexity Theory](./09-complexity-theory/)** - P vs NP and reductions
