## Practice Exercises

### Applications Created

1. **exercise-01-time-complexity.py** - Demonstrates different time complexities (O(1), O(log n), O(n), O(n log n), O(n²)) with practical examples and timing measurements.

2. **exercise-02-space-complexity.py** - Shows different space complexities (O(1), O(n), O(n²)) and demonstrates how recursion affects space usage.

3. **exercise-03-algorithm-comparison.py** - Compares different approaches to the same problem (iterative vs recursive, naive vs optimized) with performance analysis.

4. **exercise-04-big-o-analysis.py** - Practice problems for analyzing code snippets and determining their time/space complexity.

5. **practice-problems.md** - Collection of practice problems with solutions and explanations.

### Running the Exercises

```bash
# Run time complexity demonstration
python exercise-01-time-complexity.py

# Run space complexity demonstration
python exercise-02-space-complexity.py

# Run algorithm comparison
python exercise-03-algorithm-comparison.py

# View Big-O analysis examples
python exercise-04-big-o-analysis.py
```

### Key Concepts to Practice

- **Time Complexity:** Understanding how algorithms scale with input size
- **Space Complexity:** Analyzing memory usage of algorithms
- **Big-O Notation:** Formal way to describe algorithm efficiency
- **Algorithm Comparison:** Choosing the right algorithm for the problem
- **Trade-offs:** Understanding when to optimize for time vs space
